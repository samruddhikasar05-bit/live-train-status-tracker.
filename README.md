# Live Train Status Tracker

A Python-based backend application that simulates fetching real-time train running status.

---

## Project Overview

This project demonstrates backend logic development for a transport-based system similar to m-Indicator.  
It simulates fetching and displaying live train status information.

---

## Features

- Fetch train running status
- Display source and destination
- Show current station and next station
- Display delay information
- Structured backend architecture

---

## Technologies Used

- Python
- Requests Library
- JSON Handling

1. Install dependencies:

2. Run the application:
python app.py
